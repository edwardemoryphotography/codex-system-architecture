## Overview

**codex-system-architecture** is a TypeScript-based visual design tool for mapping, documenting, and iterating on the system architecture of the **Codex AI platform** — a neurodivergent execution and knowledge management system.

Built with StackBlitz for rapid prototyping, this repo serves as the living blueprint for how the Codex components connect: from AI agents and memory layers to automation workflows and UI shells.

## Purpose

This repo is the **design layer** for the broader Codex ecosystem:

- Define system components and their relationships
- Map data flows between AI agents, memory, and UI
- Prototype architecture changes before implementation
- Maintain a living technical spec for the platform
- Document neurodivergent execution patterns and system hooks
